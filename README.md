# Hub Total

Web estática lista para GitHub Pages.

## Qué incluye

- Buscador global
- Filtros por tipo
- Favoritos guardados en el navegador
- Datos separados en JSON por categoría
- Estructura escalable para cientos o miles de entradas

## Estructura

- `index.html`
- `styles.css`
- `app.js`
- `data/books.json`
- `data/games.json`
- `data/videos.json`
- `data/courses.json`
- `data/podcasts.json`
- `data/libraries.json`

## Publicar en GitHub Pages

1. Sube estos archivos a la raíz del repo.
2. En GitHub ve a **Settings > Pages**.
3. En **Build and deployment** selecciona:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/** (root)
4. Guarda y espera el enlace público.

## Nota

Los enlaces apuntan a fuentes gratuitas y legales. Puedes ampliar los JSON con más recursos cuando quieras.
